# The Seed Etreprise - Professional Website

A fully functional, professional website for The Seed Etreprise offering VIP Protection Services, Property Services, and Web Development solutions.

## 🌟 Features

- **Responsive Design** - Mobile-first approach, works perfectly on all devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Firebase Integration** - Real-time database and authentication
- **Admin Dashboard** - Manage contact submissions and service requests
- **Contact Forms** - Full form validation and submission handling
- **Service Pages** - Detailed information about all services offered
- **About Page** - Company information and team showcase

## 🛠️ Technology Stack

### Frontend
- React 18+
- React Router DOM
- CSS3 (Responsive, Modern, Professional)
- Font Awesome Icons

### Backend & Database
- Firebase 10+
- Firebase Authentication (Email + Password)
- Firestore Database
- Firebase Hosting

## 📋 Prerequisites

Before you begin, ensure you have installed:
- Node.js (v14 or higher)
- npm (v6 or higher)
- Firebase CLI (for deployment)
